# حراست سبحان بتن (Sobhan Beton Security — Entry/Exit App)

اپلیکیشن اندروید برای ثبت ورود و خروج **پرسنل**، **ماشین‌آلات**، **مراجعین/مشاوران** و **رانندگان** کارخانه بتن، با آیکون و برندینگ اختصاصی.

## ویژگی‌ها

- **ناوبری تب‌محور** با ۴ تب بالای صفحه: پرسنل، ماشین‌آلات، مراجعین، رانندگان.
- **پرسنل**: دسته‌بندی بر اساس دپارتمان (مدیریت، فروش، مالی و اداری، حراست، باچینگ، حمل‌ونقل، ماشین‌آلات) با هدرهای بخش‌بندی‌شده.
- **ماشین‌آلات**: ناوگان ثابت کارخانه از قبل در برنامه ثبت شده است:
  - میکسر: ۱۵ آمیکو + ۱۵ ایوکو (۳۰ عدد)
  - پمپ بتن: ۷ بدسند
  - کمپرسور تریلر: ۶ فاو + ۱ اسکانیا F
  - کامیون ۱۰ چرخ: ۶ آمیکو
  - وانت نیسان (تدارکات) و وانت پژو (تعمیرگاه)
- **مراجعین/مشاوران**: ورود با وارد کردن دستی نام + واحد مورد مراجعه در همان لحظه.
- **رانندگان**: ورود با وارد کردن دستی نام + ماشین مربوطه در همان لحظه.
- **قانون منطقی اصلی**: تا زمانی که خروج ثبت نشود، ورود مجدد همان فرد/ماشین امکان‌پذیر نیست.
- **عکس پروفایل/ماشین**: از صفحه «تنظیمات اولیه» می‌توان برای هر پرسنل/ماشین یک عکس از گالری انتخاب کرد؛ در غیر این صورت آیکون پیش‌فرض دسته نمایش داده می‌شود.
- **جست‌وجوی سریع**: هم داخل هر تب (فیلتر آنی)، هم یک جست‌وجوی کلی در همه دسته‌ها از آیکون بالای صفحه.
- **فید فعالیت اخیر**: زیر نوار وضعیت هر تب، آخرین رویداد به‌صورت «فلانی وارد شد» / «فلان ماشین خارج شد» نمایش داده می‌شود.
- **تاییدیه دو مرحله‌ای + لرزش**: هر ثبت ورود/خروج نیاز به تایید در دیالوگ دارد و با یک لرزش کوتاه (haptic) تایید می‌شود.
- **پشتیبان‌گیری خودکار**: بعد از هر ثبت ورود/خروج، یک نسخه از پایگاه‌داده در پوشه Downloads/ConcreteFactoryBackups ذخیره می‌شود (حداکثر ۲۰ نسخه اخیر نگه‌داری می‌شود).
- **گزارش و خروجی اکسل**: از آیکون «گزارش‌ها»، بازه تاریخی (پیش‌فرض: امروز) را انتخاب کرده و یک فایل `.xlsx` با انکودینگ UTF-8 و یک ردیف برای هر رویداد (نام، دسته، واحد/ماشین، نوع رویداد، تاریخ و ساعت) دریافت می‌کنید. فایل هم در Downloads/ConcreteFactoryReports ذخیره می‌شود، هم مستقیماً قابل اشتراک‌گذاری (تلگرام، ایمیل و…) است.
- ذخیره‌سازی دائمی با **Room (SQLite)**.
- طراحی متریال، پشتیبانی کامل RTL، رابط کاربری فارسی، آیکون اختصاصی اپ.

## آیکون و برندینگ

آیکون برنامه از تصویر اسب/ققنوس قرمز روی زمینه مشکی که ارسال شد ساخته شده:
- **آیکون لانچر (Adaptive Icon)**: زمینه مشکی + سیلوئت قرمز اسب (برای اندروید ۸ به بالا)، به‌همراه نسخه legacy برای نسخه‌های قدیمی‌تر.
- **لوگوی داخل برنامه**: همان تصویر در نوار بالای صفحه اصلی (Toolbar) نمایش داده می‌شود.

## نحوه باز کردن و اجرای پروژه

1. [Android Studio](https://developer.android.com/studio) (نسخه Hedgehog یا بالاتر) را نصب کنید.
2. پوشه `ConcreteFactoryApp` را با گزینه **Open** در Android Studio باز کنید.
3. صبر کنید Gradle Sync تمام شود (بار اول به اینترنت نیاز دارد).
4. یک دستگاه/شبیه‌ساز (حداقل API 24 / Android 7.0) انتخاب کرده و **Run ▶** بزنید.

### ساخت APK
```
./gradlew assembleDebug
```
خروجی در: `app/build/outputs/apk/debug/app-debug.apk`

### ساخت ابری بدون کامپیوتر
فایل `.github/workflows/build.yml` به‌طور خودکار APK را روی سرورهای گیت‌هاب می‌سازد؛ راهنمای کامل در `BUILD_WITH_GITHUB.md`.

## مجوزهای لازم
- دسترسی به فایل (SAF) برای انتخاب عکس — بدون نیاز به مجوز زمان اجرا.
- `WRITE_EXTERNAL_STORAGE` فقط برای اندروید ۹ و پایین‌تر (برای نسخه‌های جدیدتر از MediaStore استفاده می‌شود).

## توسعه‌های آتی پیشنهادی
- ورود با رمز/بیومتریک برای نگهبان قبل از هر عملیات.
- اعلان برای مواردی که بیش از X ساعت داخل مانده‌اند.
- همگام‌سازی ابری (اختیاری) برای پشتیبان‌گیری چند-دستگاهی.
