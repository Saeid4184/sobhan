# ساخت APK با گیت‌هاب (فقط با گوشی، بدون کامپیوتر)

این روش پروژه را روی سرورهای گیت‌هاب کامپایل می‌کند؛ گوشی شما فقط برای آپلود فایل‌ها و دانلود خروجی نهایی استفاده می‌شود.

## مرحله ۱: ساخت اکانت گیت‌هاب
1. مرورگر گوشی را باز کنید و به [github.com](https://github.com) بروید.
2. روی **Sign up** بزنید و یک اکانت رایگان بسازید.

## مرحله ۲: ساخت ریپازیتوری جدید
1. بعد از ورود، روی دکمه **+** بالای صفحه بزنید → **New repository**.
2. یک نام بدهید، مثلاً `ConcreteFactoryApp`.
3. آن را **Public** یا **Private** بگذارید (فرقی نمی‌کند) و **Create repository** را بزنید.

## مرحله ۳: آپلود فایل‌های پروژه
1. فایل zip که برایتان فرستادم را با یک فایل‌منیجر (Files by Google یا مشابه) **Extract/Unzip** کنید.
2. در صفحه ریپازیتوری گیت‌هاب، روی لینک **uploading an existing file** بزنید
   (یا از منو: `Add file → Upload files`).
3. از فایل‌منیجر گوشی، **تمام محتویات داخل پوشه `ConcreteFactoryApp`** (نه خودِ پوشه، بلکه فایل‌ها و پوشه‌های داخلش مثل `app`, `gradle`, `build.gradle`, `.github` و...) را انتخاب و آپلود کنید.
   - نکته مهم: پوشه `.github` باید حتماً آپلود شود چون فایل ساخت خودکار (workflow) داخل آن است. اگر فایل‌منیجر گوشی پوشه‌های مخفی (که با نقطه شروع می‌شوند) را در Extract نشان نداد، اسم پوشه `.github` را با تایپ مستقیم به مسیر بزنید یا از یک اپ آرشیو مثل **ZArchiver** استفاده کنید که پوشه‌های مخفی را هم نشان می‌دهد.
4. پایین صفحه پیام Commit را بنویسید (مثلاً "آپلود اولیه پروژه") و **Commit changes** را بزنید.

## مرحله ۴: اجرای ساخت خودکار (Build)
1. به‌محض آپلود، گیت‌هاب به‌طور خودکار ساخت را شروع می‌کند (چون فایل workflow روی push تنظیم شده است).
2. به تب **Actions** در بالای صفحه ریپازیتوری بروید.
3. یک ردیف با عنوان **Build Debug APK** می‌بینید که در حال اجراست (دایره زرد/نارنجی در حال چرخش).
4. چند دقیقه صبر کنید تا علامت آن به ✅ سبز تبدیل شود.

اگر شروع نشد یا خواستید دوباره اجرا کنید:
تب Actions → **Build Debug APK** (سمت چپ) → دکمه **Run workflow** → **Run workflow**.

## مرحله ۵: دانلود APK
1. روی همان اجرای سبزرنگ (✅) در تب Actions کلیک کنید.
2. پایین صفحه، بخش **Artifacts** را پیدا کنید.
3. روی `app-debug-apk` بزنید تا یک فایل zip دانلود شود.
4. آن zip را باز کنید؛ داخلش فایل `app-debug.apk` است.
5. روی `app-debug.apk` بزنید تا نصب شود (اگر گوشی هشدار "منابع ناشناس" داد، در تنظیمات اجازه نصب برای همان اپ/مرورگر را بدهید).

---

### اگر Build با خطا (❌ قرمز) مواجه شد
روی همان اجرا کلیک کنید، گزینه **Build debug APK** را باز کنید تا لاگ خطا را ببینید، و متن خطا را برایم کپی/پیست کنید تا کمک کنم برطرفش کنیم.
